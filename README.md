# SHJPro
山海经知识归档（Python3.7+Django2.1+SQLite）

三个主要应用：SHJ（前台展示），SiteView（网站统计，访问量、ip），UserInfo（用户注册）

主要功能：评论功能（Ajax提交），注册功能（拓展了admin-user），站内搜索（django-haystack/whoosh），echarts统计
